terminalden git push adımları : 1. git init yap 2. git config --global user.name ve user.email ekle 3. git add ile dosyaları ekle 4. git commit -m 'yapılan değişiklik' kodu ile commit yap 5. git branch -M main komutu ile main branch oluştur. 6. git remote add origin 'git yüklenecek repo' koduyla remote bağlantı kur. 7. git pull --rebase origin main ile senkronize 8. git push -u origin main komudu ile repoya dosyları gönder. daha sonra projelerde çalışırken pull ve push yaparak projeye devam et.
